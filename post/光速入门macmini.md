## MacOS 创建快捷键

| 修饰键  | 相当于`windows`键 |
| ------- | ----------------- |
| command | windows           |
| option  | alt               |
| control | ctrl              |
| fn      | 功能键            |

| 快捷键                      | 功能                                     |
| --------------------------- | ---------------------------------------- |
| command + c                 | 复制                                     |
| command + v                 | 粘贴                                     |
| command + x                 | 剪切                                     |
| command + z                 | 撤销                                     |
| command + shift + z         | 重做                                     |
| command + a                 | 全选                                     |
| command + m                 | 最小化窗口,不能能通过`command + tab`切换 |
| command + h                 | 隐藏窗口                                 |
| control + command + f       | 全屏切换                                 |
| command + s                 | 保存                                     |
| command + q                 | 退出                                     |
| command + w                 | 关闭窗口                                 |
| command + o                 | 打开文件                                 |
| command + p                 | 打印                                     |
| command + tab               | 切换应用程序                             |
| command + tab               | 切换应用程序                             |
| command + space             | 打开 Spotlight 搜索                      |
| command + shift + 3         | 截取整个屏幕                             |
| command + shift + 4         | 截取选定区域                             |
| command + shift + 4 + space | 截取选定区域                             |
| command + shift + 5         | 截取屏幕并显示选项                       |
| option + command + esc      | 强制退出应用程序                         |
| command + ,                 | 打开应用程序的偏好设置                   |
| command + shift + n         | 在 Finder 中创建新文件夹                 |
